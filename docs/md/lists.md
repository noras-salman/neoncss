## Lists

#### Vertical

```html
<ul class="list">
  <li>List item <span class="label">label</span></li>
  <li>List item <span class="badge">122</span></li>
  <li>List item</li>
  <ul class="list sublist">
    <li>List item</li>
    <li class="active">Active list item</li>
    <li>List item</li>
    <li>List item</li>
    <ul class="list sublist">
      <li>List item</li>
      <li>List item</li>
    </ul>
  </ul>
</ul>
```

#### Horizontal

```html
<ul class="list horizontal">
  <li>List item</li>
  <li>List item</li>
  <li>
    <a href="#">List item <span class="badge">22</span></a>
  </li>
  <li>List item</li>
  <li>
    <div class="dropdown">
      <span>Dropdown</span>
      <ul class="list">
        <li>List item <span class="label">label</span></li>
        <li>List item <span class="badge">122</span></li>
        <li>List item <span class="badge sufix">122</span></li>
        <li>List item</li>
      </ul>
    </div>
  </li>
</ul>
```
