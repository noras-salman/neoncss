## Getting started

##### From CDN:

CSS:

```
https://cdn.jsdelivr.net/gh/noras-salman/neoncss/release/latest/css/neoncss.min.css

```

JS:

```
https://cdn.jsdelivr.net/gh/noras-salman/neoncss/release/latest/js/neoncss.min.js
```

##### Template

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
    <link
      rel="stylesheet"
      href="https://cdn.jsdelivr.net/gh/noras-salman/neoncss/release/latest/css/neoncss.min.css"
    />
  </head>

  <body>
    <script src="https://cdn.jsdelivr.net/gh/noras-salman/neoncss/release/latest/js/neoncss.min.js"></script>
  </body>
</html>
```
